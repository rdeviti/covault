# Testbed Setup Scripts (MPI-SWS Infrastructure)

This folder contains scripts specifically designed for setting up the testbed environment within the MPI-SWS infrastructure. This includes configurations for Intel and AMD machines, SEV-SNP support, disable frequency scaling, and specific networking configurations. These scripts are tailored to this particular infrastructure. However, feel free to adapt and modify them to suit your own specific use case and environment if needed.

The folder also includes some scripts to format the Python and Cpp code.
